JSON

  • JavaScript object notation（JavaScript对象表示法）
  • JSON是轻量级的文本数据交换格式
  • JSON独立于语言，具有自我描述性，更易理解

json语法规则

  • 数据由键（key）/值（value）描述，由逗号分隔
  • 大括号代表一个完整的对象，拥有多个键/值对
  • 中括号保存数组，多个对象之间使用逗号分隔
